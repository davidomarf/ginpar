CLI Commands
============

This is a list of the available CLI commands for Ginpar.

This page contains the same information you'd get if you run

.. prompt:: bash

  ginpar --help

and then,

.. prompt:: bash

  ginpar COMMAND --help

for every command available.

  :prog: ginpar
